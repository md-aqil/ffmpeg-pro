# Image Studio Redesign - Pipeline-Based Approach

## Current State Analysis

The current Image Studio uses a **tab-based one-tool-at-a-time** interface:
- 7 separate tabs: Convert, Resize, Crop, Effects, Pro, Optimize, Stamp
- User selects one tab, configures options, applies, then selects another tab
- Each operation is separate - requires re-uploading file each time
- Not intuitive for workflows requiring multiple operations

## User Request

Redesign Image Studio to apply **all effects in one go** (convert, resize, crop) and export the final result as a single pipeline.

---

## Stitch Design Prompt (Enhanced)

A professional image editing workspace with a pipeline-based workflow interface for applying multiple transformations in sequence.

**DESIGN SYSTEM:**
- Platform: Web, Desktop-first
- Palette: Primary (#6366f1 - indigo/violet gradient), Background Light (#ffffff), Background Dark (#09090b), Surface Dark (#18181b), Text Primary (#f4f4f5), Text Secondary (#a1a1aa)
- Styles: Rounded corners (0.375rem), Glassmorphism panels with subtle borders, Card shadows, Premium gradients (indigo to purple)

**PAGE STRUCTURE:**

1. **Header:** Minimal navigation bar with glass effect, "Image Studio Pro" title, dark/light theme toggle

2. **Main Workspace Area:** Two-column layout
   - **Left Panel (30%):** Transformation Pipeline Builder
     - Vertical stack of transformation cards (each is an enabled/disabled toggle)
     - Drag handles to reorder transformations in chain
     - Add new transformation button
     - Each card has: enable toggle, transformation type icon, name, expand to show params, delete button
     - Transformation types: Resize, Crop, Rotate, Effects, Brightness/Contrast, Saturation, Format Convert, Watermark, Optimize
   - **Right Panel (70%):** 
     - Large image preview area (current step in pipeline)
     - Before/After comparison slider
     - Upload area when no image loaded
     - Bottom action bar: "Apply Pipeline" primary button, Reset, Export Options

3. **Right Sidebar (20%):** 
   - Image metadata panel
   - Preset save/load dropdown
   - Export format selector (PNG, JPG, WEBP, GIF)
   - Quality slider

4. **Bottom Bar:**
   - Pipeline progress indicator (step X of Y)
   - Current transformation name
   - Download Final button (prominent)
   - Log/status area

**OUTPUT COMPONENTS:**
- Text Description: Pipeline-based image editor with transformation stack, reorderable effects chain, one-click apply all, before/after preview
- Suggestions: Add keyboard shortcuts for reordering, Show estimated file size change, Add undo/redo for pipeline steps
